static ExternalPotential.CustomLJ93(sigma_sf, epsilon_k_sf)#

  • sigma_sf (numpy.ndarray[float]) Solid-fluid interaction diameters.

  • epsilon_k_sf (numpy.ndarray[float]) Solid-fluid interaction energies.
